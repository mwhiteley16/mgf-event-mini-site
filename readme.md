## Mini Site Readme

This child theme is a template for the Mobile Growth Fellowship mini-sites.

## Requirements

  - Genesis FrameWork
  - Advanced Custom Fields Pro Plugin
  - Mobile Growth Functionality Plugin (see other repository)
